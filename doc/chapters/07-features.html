7.2 已实现模块

7.2.1 WebSocket Hub

位置：internal/ws/hub.go

能力：

  • 多客户端连接管理（register / unregister）
  • 事件广播到所有连接的客户端
  • Ping/Pong 心跳保活（30s 间隔）
  • 256 条消息缓冲队列

7.2.2 Event System

位置：pkg/event/event.go

能力：

  • 统一 Event 结构体（event_id, task_id, agent_id, step_index, type, data）
  • 18 种事件类型：生命周期、LLM 交互、工具调用、Agent 步骤、控制、系统
  • 自动生成唯一 ID（crypto/rand）
  • 毫秒级时间戳

7.2.3 SQLite 持久化

位置：pkg/db/database.go

能力：

  • 6 张表的 Schema 定义
  • CREATE TABLE IF NOT EXISTS 幂等初始化
  • 纯 Go 实现（modernc.org/sqlite，无 CGO）
🔶 待完善
数据库初始化函数 db.Init() 已定义但 Server 启动时未调用，Phase 1 启动时需接入。

7.2.4 Tool Registry 框架

位置：internal/tool/registry.go

能力：

  • Tool 接口定义（Name, Description, Parameters, Execute）
  • Registry 实现（Register, Execute, List, ToJSON）
  • 执行结果结构化返回
🔶 待实现
3 个内置工具（run_shell, write_file, read_file）Phase 1 实现。
