3. 技术栈

3.1 后端（Go）

🟢 运行时

Go 1.25

标准库 net/http，无 MVC 框架。保持最小依赖。

🔌 WebSocket

gorilla/websocket v1.5

成熟的 WebSocket 实现，支持自定义 upgrader 和消息 pump。

🗄️ 数据库

modernc.org/sqlite v1.53

纯 Go SQLite，无需 CGO 编译，单文件部署。

📦 其他

google/uuid gorilla/websocket

UUID 生成、JSON 处理使用标准库 encoding/json。
